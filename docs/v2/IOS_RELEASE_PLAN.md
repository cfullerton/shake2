# iOS Release Plan

## MVP Release Candidate

Release scorekeeper-only first if quality is good.

## App Store Positioning

Initial description should emphasize scoring real-world Texas 42 games, marks tracking, undo, game history, and no account required.

## Pre-Submission Checklist

- App icon
- Launch screen
- Privacy policy
- App Store screenshots
- No broken links
- No debug screens
- No placeholder copy
- Works offline
- Handles app restart
- Tested on small and large iPhones
- Accessibility labels on important controls

## Privacy

For scorekeeper-only, no account required and local storage only. When multiplayer is added, update privacy policy for account, gameplay, and room data.
