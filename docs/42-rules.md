# Texas 42 Rules Reference

## Standard Game
- 4 players
- Partners sit opposite
- Double-six set (28 dominoes)
- 7 dominoes each
- 7 tricks

## Count Dominoes
0-5 = 5
1-4 = 5
2-3 = 5
5-5 = 10
4-6 = 10

Total count points: 35
Total trick points: 7
Total hand value: 42

## Engine Priorities
1. Legal play validation
2. Trump handling
3. Trick winner determination
4. Bid evaluation
